# Clustering Analysis using K-Means and DBSCAN

## Overview
This project implements unsupervised machine learning techniques using K-Means and DBSCAN clustering algorithms on:
- Mall Customers Dataset
- Iris Dataset

The project demonstrates clustering, visualization, silhouette analysis, and customer segmentation.

---

## Features
- Data preprocessing using StandardScaler
- K-Means clustering
- DBSCAN clustering
- Cluster visualization using Matplotlib
- Silhouette score evaluation
- Customer cluster profiling

---

## Technologies Used
- Python
- NumPy
- Pandas
- Matplotlib
- Scikit-learn

---

## Dataset Details

### 1. Mall Customers Dataset
Features used:
- Annual Income (k$)
- Spending Score (1-100)

### 2. Iris Dataset
Features used:
- Sepal Length
- Sepal Width

---

## Algorithms Used

### K-Means
A centroid-based clustering algorithm that partitions data into K clusters.

### DBSCAN
A density-based clustering algorithm capable of detecting arbitrary-shaped clusters and outliers.
